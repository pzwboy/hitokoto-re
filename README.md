# Hitokoto-RE（非官方“一言”）语句库

基于[一言开源社区官方句子包](https://github.com/hitokoto-osc/sentences-bundle)v1.0.431

由于官方一言因各种原因，导致多年无新句子入库，于是本项目便诞生了。

目前仅实现了语句接口复刻

`https://hitokoto.pzwboy.top`

[官方参数](https://developer.hitokoto.cn/sentence/)可用
